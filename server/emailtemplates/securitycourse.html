Bootcamp Description:

This is a fun and hands-on ONSITE “Learn by doing” course (300+ hours) that prepares students to learn real world skills involving Cyber Security and Software Engineering. Students who finish this course will be prepared for a job in Cyber Security. This program consists of hands-on labs learning: defense and offense, self-study, system administration, DevOps, Full-Stack web application development, collaboration, team building events, professional speakers, conferences, and more! Overall you will learn the skills to help keep a company secure from advanced threats.

Who is this Bootcamp for?

Individual who work in IT and want to Level-Up to have a career, want more “hands-on” training to become technical, want to pursue a career in the field in Cyber Security, want to learn the latest Security Technologies in the Information Security Industry, or want to learn how to use a modern Tech Stack for Full-Stack DevOps. This Bootcamp is available to everyone.

How to Apply & Enroll:

  1. Apply on Bootcamp on website https://www.jidoka.us for email notifications and updates!

  2. View Syllabus on https://www.jidoka.us/bootcamp

  3. Pay for $100 Security Deposit to RSVP on Eventbrite ( Refundable 30 days before start of Bootcamp. 40 person max capacity )

  4. Max Capacity for this Bootcamp is exactly 24 students.

  5. Interview with Instructor coordiated via email/phone.

  6. After interview you will be enrolled into the Bootcamp.

  7. Download Syllabus: Jidoka_Cyber_Security_Bootcamp_Batch_1_Syllabus_2018.pdf

  8. First Day of Bootcamp is May 7 (Location still being determinted).

  9. Monthly Payment Due (May 14, June 24, & July 22) ; Cost is determined by Program Type and Deposit is deducted
